# Sounds Directory

This directory contains audio files for the TaskTimer app.

## Required Files

### siren.mp3
Place your siren/alert sound file here as `siren.mp3`.

The app uses this sound for 30-minute interval alerts. If the file is missing, the app will fall back to a generated beep sound using Web Audio API.

### Recommended Sound Characteristics
- Duration: 1-3 seconds
- Format: MP3
- Clear, attention-grabbing alert tone
- Not too harsh or jarring

### Free Sound Resources
You can download free alert sounds from:
- [Freesound.org](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/notification/)
- [Zapsplat](https://www.zapsplat.com/)
